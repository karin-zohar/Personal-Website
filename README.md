## ✨ Features

- **AI-Powered Personal Assistant** – Answers visitor questions intelligently using OpenAI API.
- **Contact Form with Email Notifications** – Powered by Nodemailer, allowing users to contact me directly. 
- **Customized Experience** – users can select the desired website theme and language. 

## ✅ Development Standards

- **Interactive UI** – Built with React and Ant Design for a smooth and responsive user experience. Optimized for desktop, tablet, and mobile devices.

- **Serverless Deployment** – Hosted on Vercel for fast, scalable, and zero-maintenance deployment.

- **Type-Safe Development** – Entire project developed in TypeScript for maintainability and reliability.

- **Global State Management** – Efficient state handling using Zustand.
